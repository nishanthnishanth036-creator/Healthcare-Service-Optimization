 🏥 Healthcare Service Optimization

 📌 Project Overview
This project analyzes 8,000+ healthcare records using SQL and Power BI to identify patient admission trends, hospital performance, treatment costs, and resource utilization.

 🛠 Tools Used
- SQL (MySQL)
- Power BI
- DAX

 📂 Dataset
- 8,000+ Healthcare Records

 📊 SQL Concepts Used
- INNER JOIN
- GROUP BY
- Aggregate Functions
- CASE Statements
- CTEs

 📈 Dashboard KPIs
- Total Revenue
- Total Bed Days
- Total Patients
- Average Length of Stay
- High-Cost Patients

 💡 Key Insights
